## 📚 Challenge Categories

| Category | Challenges | Difficulty |
|----------|-----------|------------|
| **Web Exploitation** | SQLi, XSS, File Upload, IDOR | Easy-Hard |
| **Cryptography** | Ciphers, Hashing, Encoding | Easy-Medium |
| **Forensics** | Memory dumps, PCAP analysis, Steganography | Medium-Hard |
| **OSINT** | Information gathering, Social media analysis | Easy-Medium |
| **Reverse Engineering** | Binary analysis, Decompilation | Medium-Hard |
| **Privilege Escalation** | Linux/Windows privesc scenarios | Medium-Hard |

## 🚀 Quick Start

Each challenge includes:
- Challenge description and objectives
- Hints (progressive disclosure)
- Solution walkthrough (spoiler-protected)
- Learning outcomes

```bash
git clone https://github.com/OluwagaAy/CTF-Challenges.git
cd CTF-Challenges
```
